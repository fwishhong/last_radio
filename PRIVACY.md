# Privacy Policy

Last Radio: Old Stadium Watch (旧体育馆守夜) — built with Godot Engine 4.3.

Last updated: 2026-06-19

## What we collect

We collect **nothing on our servers**. The game runs entirely offline once
installed. There is no analytics SDK, no telemetry beacon, no crash reporter,
and no account system.

The only data the game stores is on **your own computer**, in the
platform's user data directory:

| Platform | Path |
|---|---|
| Windows | `%APPDATA%\Godot\app_userdata\Last Radio v2\` |
| macOS   | `~/Library/Application Support/Godot/app_userdata/Last Radio v2/` |
| Linux   | `~/.local/share/godot/app_userdata/Last Radio v2/` |

Files in that directory:

- `saves/last_radio_chapter_01_*.json` — your game progress
- `settings.json` — your language, audio, and window preferences
- `shader_cache/`, `vulkan/` — Godot render cache (auto-managed)

These files are local to your machine and never transmitted.

## When you play on Steam (Steam release only)

If you bought the game on Steam, Steam itself collects the data described in
[Steam's Privacy Policy](https://store.steampowered.com/privacy_agreement/).
That includes:

- Your Steam user ID
- Your playtime and achievements
- Crash dumps (only if Steam's "send crash data" setting is enabled on your
  client)

We do not have access to anything beyond what Steam's partner API exposes
(achievements, cloud save sync, basic stats). We do not share that data with
any third party.

## Cloud saves (Steam release only)

If you opt in to Steam Cloud synchronization, your save files are uploaded to
Steam's servers so you can play across machines. Cloud data is encrypted in
transit and at rest by Steam.

You can disable cloud sync from the game's settings menu, in which case
saves remain local only.

## Children's privacy

The game is not directed at children under 13. We do not knowingly collect
any data from children. Steam itself has its own age-gating; we honor
whatever age gate is set in your Steam client.

## Your rights

You can:

- **Delete your local saves** at any time by uninstalling the game or
  deleting the user data directory listed above.
- **Delete your cloud saves** from the Steam client.
- **Request a copy of any data we hold** — note that we hold nothing
  directly; all data lives on Steam's infrastructure.

## Changes to this policy

If we change this policy, we'll note the change in the patch notes. The
last-updated date above will also change.
